HTML

In order to create a website, you should know HTML - the basic technology used to define the structure of a web page. HTML is used to indicate that your web content should be understood as paragraphs, lists, headers, links, images, multimedia players, forms, or one of the many other elements available or new elements you define.

After learning HTML, you can move on to more advanced topics such as:

CSS, and how to decorate HTML with it (eg: change your text size and font, add borders and shadows, design your page as a multi-column layout, add animations and other visual effects.)

JavaScript, and how to use it to add dynamic functionality to web pages (eg: find your address and draw it on a map, make UI elements appear or disappear when a button is triggered, store user data locally on their computer, and more .)

Lesson Plan

1.Introduction to HTML

This module will pave the way for you to get used to some important concepts and syntax, looking at how to apply HTML to text, how to create hyperlinks, and how to use HTML to construct a web page.

2.Multimedia and embedding

This module takes you through an exploration of how to use HTML to include multimedia information in your web pages, including how to include images in various ways, and how to embed video, audio, and even other complete web pages.

3.HTML table

Representing tabular data on web pages in an understandable and accessible way is a challenge. This module includes basic table markup, plus more complex features such as implementing headings and summaries.
